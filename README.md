# Got_Brains: Math Trivia App

## Project Installation

1. Install dependencies

```bash
$ cd GotBrains
$ npm install
```

2. Run webpack bundler

- Development

  ```bash
  $ npm run dev
  ```

  - Production

  ```bash
  $ npm run build
  ```

## Run the application

1. Start server

   - Development

   ```bash
   $ npm run dev:server
   ```

   - Production

   ```bash
   $ npm run server
   ```

2. Go to http://localhost:3000/

## To Run Tests

```bash
$ npm run test
```
